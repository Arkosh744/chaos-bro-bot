# Chaos Bro Bot — Roadmap

## v0.1 — MVP (done)
- [x] Grounding-техники (30+ из пула)
- [x] Хаос-задания (47 из пула + генерация claude)
- [x] Рандомайзер решений через claude
- [x] Свободный чат с трикстером
- [x] Память: SQLite, summary каждые 20 сообщений, контекст в каждый запрос
- [x] Thinking-анимация 🤔 → 🤔. → 🤔..
- [x] Inline кнопки "Ещё" / "Другое" (edit вместо нового сообщения)
- [x] Рандомные имена при /start (Геральт из Пятёрочки, Мерлин Бухой...)
- [x] Scheduler: рандомные пинги 2-6ч (цитаты из игр, grounding, трикстер)
- [x] Тесты: unit + integration

## v0.2 — Personality
- [ ] Голосовые сообщения: Groq Whisper API → текст → claude → ответ
- [ ] Бот отвечает голосовухой (TTS через edge-tts или piper)
- [ ] Стикеры: бот присылает рандомный стикер к ответу
- [ ] Мемы/гифки: генерация или подбор из пула
- [ ] Больше grounding-техник (дыхательные таймеры, body scan)

## v0.3 — Social
- [ ] Групповой чат: реагирует на упоминания, рандомно влезает
- [ ] Реакции на стикеры/фото ("чёт ты грустный на фотке")
- [ ] Поддержка нескольких юзеров с разными summary
- [ ] Rate limiter (чтоб не разориться на claude)

## v0.4 — Analytics
- [ ] Ежедневный/еженедельный дайджест ("за неделю ты 5 раз ныл про работу")
- [ ] Mood tracking по сообщениям (без явного ввода)
- [ ] Streak: сколько дней подряд общаешься с ботом
- [ ] Статистика: топ тем, частота сообщений по дням

## v1.0 — Production
- [ ] Docker image
- [ ] Systemd unit для автозапуска
- [ ] Graceful shutdown с сохранением состояния
- [ ] Health check endpoint
- [ ] Метрики (prometheus)
- [ ] CI/CD (GitHub Actions: lint, test, build)
