# Chaos Bro Bot

Дерзкий друг-трикстер в Telegram. Не коуч, не терапевт, не AI-ассистент. Помогает жить в моменте через сарказм, абсурд и контролируемый хаос.

## Кнопки

| Кнопка | Что делает |
|--------|-----------|
| **Очнись** | Grounding-техника из пула 30+ упражнений. Inline-кнопка "Ещё" подгружает следующую в то же сообщение |
| **Ебани куба** | Рандомное хаос-задание (47 заготовок + генерация через Claude). "Другое" — новое задание в том же сообщении |
| **Кинь кости** | Задай вопрос с `?` — бот решит за тебя. Однозначно и дерзко |
| **Судьба** | Рандомное трикстерское предсказание через Claude |
| **Дыши** | Дыхательный таймер: 3 раунда (вдох 4с, задержка 4с, выдох 4с). Анимация через редактирование сообщения |

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие с рандомным фэнтези-именем (Геральт из Пятёрочки, Мерлин Бухой, Голлум с Авито...) |
| `/capsule N текст` | Капсула времени — бот доставит сообщение тебе через N дней |

## Чат

- **Свободный текст** — трикстер отвечает через Claude с характером
- **Вопрос с `?`** — рандомайзер решений вместо ответа
- **Голосовое сообщение** — транскрипция через Groq Whisper, затем ответ от Claude

## Личность

| Фича | Как работает |
|------|-------------|
| **Режим сна** | 23:00-09:00: без Claude, ответы из пула ("zzz...", "сплю отъебись") |
| **Пасхалки** | зуг-зуг, 42, кабан, gg, F, ку — мгновенные ответы |
| **Настроение по времени** | Утро=сонный, день=дерзкий, вечер=философский, ночь=мистический |
| **Альтер-эго** | ~1/7 шанс в день, на 24ч меняет личность (Философ, Гопник, Аристократ, Пират, Бабка, Йода) |
| **Обида** | Если не писал >24ч — бот обижается |
| **Торг** | 20% шанс: "Отвечу, но сначала сделай 5 приседаний" |
| **Лут** | Каждые 10 сообщений выпадает подарок |

## Память

- Полная история сообщений в SQLite
- Summary контекста обновляется каждые 20 сообщений через Claude
- Каждый вызов Claude получает summary + последние 5 сообщений
- Детектор паттернов: замечает повторяющееся поведение, троллит

## Планировщик

| Событие | Расписание |
|---------|-----------|
| **Рандомные пинги** | Каждые 2-6ч: игровые цитаты (40%), grounding (30%), подъёбка с контекстом (30%) |
| **Утренний чекин** | "Как ты от 1 до 10?" с inline-кнопками, ответ по настроению |
| **Вечерний чек** | "Что хорошего за день?" |
| **Недельный дайджест** | Воскресенье вечером: статистика + roast |
| **Доставка капсул** | Проверка каждый час |

## Быстрый старт

```bash
git clone https://github.com/Arkosh744/chaos-bro-bot.git
cd chaos-bro-bot

go mod tidy

cp config.example.yaml config.local.yaml
# Заполни config.local.yaml

go build -o bot ./cmd/bot/
./bot
```

### Требования

- Go 1.26+
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) (`claude` в PATH)
- Telegram Bot Token ([@BotFather](https://t.me/BotFather))
- (опционально) Groq API Key — для транскрипции голосовых

### Конфиг

```yaml
telegram:
  token: "YOUR_BOT_TOKEN"
  owner_id: 123456789

claude:
  model: "haiku"
  timeout: 120s

scheduler:
  enabled: true
  min_hour: 9
  max_hour: 22

storage:
  db_path: "trickster.db"

groq:
  api_key: "YOUR_GROQ_KEY"  # для голосовых сообщений
```

Поддерживает `$ENV_VAR` в значениях. Fallback: `TELEGRAM_TOKEN` из окружения.

## Архитектура

```
Сообщение/голос → [Groq Whisper] → SQLite → summary + last 5 → claude -p → ответ → SQLite
                                                                                ↓
                                                  каждые ~20 сообщений → обновляет summary
                                                  детектор паттернов → roast если надо
```

```
cmd/bot/main.go              — entrypoint
internal/
├── bot/                      — telebot handlers, thinking-анимация
├── claude/                   — обёртка claude -p (haiku)
├── config/                   — YAML конфиг + env expansion
├── features/                 — всё интересное:
│   ├── grounding.go          — 30+ техник заземления
│   ├── chaos.go              — 47 хаос-заданий
│   ├── breathing.go          — дыхательный таймер
│   ├── alterego.go           — альтер-эго (6 личностей)
│   ├── memory.go             — контекст + summary
│   ├── patterns.go           — детектор паттернов
│   ├── digest.go             — недельный дайджест
│   ├── prompts.go            — системные промпты
│   ├── quotes.go             — игровые цитаты
│   ├── randomizer.go         — рандомайзер решений
│   ├── trickster.go          — торг, обида, лут, пасхалки
│   └── fallback.go           — пул ответов для сна/оффлайна
├── groq/                     — Groq Whisper API
├── scheduler/                — пинги, чекины, капсулы, дайджест
└── storage/                  — SQLite: messages + context_summary
```
